## What This Is

This is the interactive UI mockup for the STRUCZ™ platform — the design reference that Jamie builds from in Blazor C#. It is **not** the production codebase. It is an HTML/CSS/JS prototype that demonstrates:

- Global Shell — navigation, header, ZED panel, presence indicators
- Command Dashboard — live metrics, GP progress, requirements feed, ZED overnight activity  
- Contact Manager — company list with search/filter, company record with timeline
- ZED Panel — ZED Needs You queue, morning briefing, Ask ZED

## Navigation

Click any item in the left sidebar to navigate. Click any company row to open its record. Use the search bar to filter companies. Click ZED in the header to toggle the ZED panel.

## Production Build

The production application is built by Jamie Cunningham in:
- **Framework:** .NET MAUI Blazor Hybrid
- **Backend:** C# .NET 8 Web API
- **Database:** PostgreSQL
- **AI:** ZED™ via LiteLLM → Anthropic Claude
